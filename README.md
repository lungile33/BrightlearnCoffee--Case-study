## Bright coffee shop: Accelerating Progress - Sales Performance Analysis (Jan - June 2023)

## Problem statement 
The coffee shop has experienced a measurable decline in sales performance over the past six months, resulting in reduced revenue generation. This downward trend indicates potential inefficiencies or underlying issues within business operations, customer behavior, or market conditions. The objective of this analysis is to identify the key drivers contributing to the decline in revenue and overall performance.

### Overview
This project provides a comprehensive analysis of Bright Coffee Shop's Sales performance for the period January to June 2023.

##Aim

The aim of this project is to conduct a comprehensive exploratory and descriptive data analysis to uncover actionable insights into the factors influencing the decrease in revenue. The findings will support data-driven decision-making for the CEO and inform strategic interventions to improve business performance.

### To achieve the project aim, the following steps were undertaken:

-Project Planning and Design
-Developed a structured analytical approach and project workflow using Canva to outline key milestones, deliverables, and timelines.
-Ideation and Requirements Gathering
-Conducted collaborative brainstorming sessions using Miro to define business questions, hypotheses, and key performance indicators (KPIs).
-Data Ingestion and Processing

### Tools and Technologies used
- Miro: Used for project planning and workflow mapping
- Excel: Used for data cleaning, preparation, and pivot table analysis
- SQL: Used for querying sctructured data and extracting insights
- Cnva for designing and presenting the final presentation
-Utilized Databricks to perform data extraction, transformation, and loading (ETL), ensuring data quality, consistency, and readiness for analysis.
-Data Analysis and Interpretation (added for completeness)
-Performed exploratory data analysis (EDA) to identify trends, patterns, and anomalies in sales data over the six-month period.
Insight Generation and Reporting (added for completeness)
Translated analytical findings into business insights and recommendations to address the identified causes of revenue decline.
